HOW TO USE:

Set Directory:
Simply input directory of text files that you may want to load into the calculator. Valid textfiles include letters that correspond to actions for the
calculator.

4 Function Calculator:
Input numbers to add numbers to the display. Use the operators +, -, * and / for addition, subtruction, multiplication and division respectively.
Other operators include:
c - Clear display
! - Turn current number into its negative
s - Sqrt the current number
m(1 - 9) - Store current number into 1 of 9 memory slots
r(1 - 9) - Load a number from one of the memory slots
